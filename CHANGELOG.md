# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-04

### Added

- Initial release
- `RequirementsChecker` - Main requirements validation class
- `RequirementsConfig` - Configuration builder from array/file
- `RequirementsServiceProvider` - Service provider integration
- `RequirementsErrorHandler` - Admin notices and wp_die handling
- `PhpVersionRequirement` - PHP version validation
- `WordPressVersionRequirement` - WordPress version validation
- `PhpExtensionRequirement` - PHP extension validation
- `PluginRequirement` - Plugin dependency validation
- `MultisiteRequirement` - Multisite compatibility validation
- Full documentation and examples
